TA(Travel Assistant) System

Project Description

    This project is supported by the National Natural Science Foundation of China. The project's full name is Internet-facing Mining and Cross-media Search Engine. The data of TA Knowledge Graph based on Beijing 1819 tourist attractions, and a semantic search system impelemented based on the Knowledge Graph, then return the tourists knowledge card based on the users' input key words. TA(Travel Assistant) system also supports image retrieval, users can upload attractions pictures to retrieval the results.

Related Tecnologies

  • Use crawlers to download the factual information and pictures of the sites such as Beijing tourism, Wikipedia and Flickr.
  • Combined with Semantic Web tecnology,for example RDF(Resource Description Framework),to build knowledge graph of Beiing tourists.
  • Design and implement the TA system based on Jena,Lucene,Hadoop.
  • Use LSH(Locality Sensitive Hashing) algorithm to implement image retrieval based on the upload pictures by the users.
