Commodity Classification(Aisino)

Project Description

    This project mainly focus on commodity classification based on the data of tax invoice. The important data contaions hydm, hwmc and spbm meaning 行业代码, 名称, 商品编码 respectively. There are 20737 samples which has spbm.We must label the rest data dosn't has spbm.

  • parsing unicode hwmc to single charactor and generating the charactors to bi-grams
  • Using Naive Bayes model to classify the hwmc which without the spbm.
  • Appending extra corpus to train a word embedding with word2vec
  • Combining the single charactor vector,bi-gram vector and hwmc word embedding
  • Blending several classifier, such as LR, SVM, Naive Bayes

Enterprise Portrait(Aisino)

Project Description

    This project mainly focus on Enterprise Portait based on the data of tax invoice. Currently we had developed several tags of the enterprise, such as the rapid growth， supply chain core business, cycloid type enterprises and so on.

  • Spark big data analysis
  • Calculate the periodicity of enterprise using AutoCorrelation
  • Linear Regression and EigenVector Centrality

Travel Assistant(CAS)

Project Description

    This project is supported by the National Natural Science Foundation of China. The project's full name is Internet-facing Mining and Cross-media Search Engine. The data of TA Knowledge Graph based on BeiJing 1819 tourist attractions, and a semantic search system implemented based on the Knowledge Graph, then return the tourists knowledge card based on the users' input key words. TA(Travel Assistant) system also supports image retrieval, users can upload attractions pictures to retrieval the results.

Text classification with Python

Project Description

    This project mainly focus on the topic of text classification, including the english and chinese text. Several open projects have been used, such as Scikit-Learn, NLTK, Gensim, Numpy, Pandas, Matplotlib.

  • Bag-of-Words and TFIDF as the representation of the text respectively
  • SVM, Logistic Regression, Multiple Layer Perception(ANN), Random Forest as the Classifier
  • Plot the results using matplotlib and stanford seaborn
  • Scikit-Learn Machine Learning Pipeline, GridSearch, Cross-Validation...
